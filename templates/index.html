FAQ

  • How can I join meeTOmove?

    If you are a player create a new user profile by clicking here and filling the sign up form. If you are a sport club click here instead.

  • How can I join an existing event?

    Select the event you are looking for from here and join it by clicking the Join Event button.

  • How can I create an event?

    Reach the dedicated page here and fill the Add Event form.

  • How can I leave a Review?

    From your events select an event and click the Leave Review button. Then select a player and fill the Create Review form.
